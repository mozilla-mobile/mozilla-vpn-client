# Security Policy

Mozilla VPN participates in the [Mozilla Client Security Bug Bounty Program](https://www.mozilla.org/en-US/security/client-bug-bounty/). The program is designed to encourage security research into Mozilla's websites and services and to reward those who find unique and original security bugs in Mozilla's products and web services. That website has details on the program, including submission guidelines and program rules.

<br />

## Reporting a security bug

**Bug bounty hunters:**
Please submit security bug reports for the Mozilla Client Security Bug Bounty Program via [Bugzilla](https://bugzilla.mozilla.org/form.client.bounty). It is recommended that you first read the program's guidelines and FAQ, which are linked above.

**Everyone else:** 
Please email <a href="mailto:security@mozilla.com">security@mozilla.com</a> with the details.
